# BME160-final-Project
This research project is based on enzyme kinetics, specifically for plotting enzyme activity. The data collected is from Seth Rubin, a PI at UCSC’s research project, “Structural basis of divergent cyclin-dependent kinase activation by SPY1/RINGO Proteins.” This research focuses on Cyclin bound Cyclin-dependent kinase and how proteins such as SPY1/RINGO can cause tumorigenesis, especially if there are protocols on line. There was data within the article that fixates on the enzyme activity on the wild type and mutated type of Cyclin bound Cyclin-dependent Kinase through the usage of Michaelis-Menten. Our program creates a plug in tool so the data can be fairly expedited for those that are time constrained and limit the amount of manual labor from these equations. The tool, which gives the Michaelis-Menten plotting already, will also effectively give the Line Weaver Burke plot as well. This plotting program will be efficient for those in enzymology research, and accessible for further research in other fields of study.
